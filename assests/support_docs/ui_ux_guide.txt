================================================================================
5. ERROR MESSAGES & VALIDATION
================================================================================

Error Message Display:
- Color: #dc3545 (Red)
- Font Size: 0.9rem
- Margin Top: 5px
- Display: Initially hidden (display: none)
- Show on validation failure

Error Message Text:
- Name Error: "Please enter your name"
- Email Error: "Please enter a valid email"
- Address Error: "Please enter your address"
- Empty Cart: "Your cart is empty!" (alert popup)

Discount Code Messages:
Success (Valid Code):
- Text: "✅ 15% discount applied!" or "✅ 20% discount applied!"
- Color: #28a745 (Green)

Error (Invalid Code):
- Text: "❌ Invalid discount code"
- Color: #dc3545 (Red)

Error (Empty Code):
- Text: "Please enter a discount code"
- Color: #dc3545 (Red)

Validation Rules:
- Email Format: Must contain "@" and valid domain
- Name: Minimum 1 character (non-empty)
- Address: Minimum 1 character (non-empty)
- Real-time validation on "Pay Now" button click
- All errors must be displayed in red text below the input field

8. ACCESSIBILITY REQUIREMENTS
================================================================================

WCAG 2.1 Level AA Compliance:

Color Contrast:
- Text on white background: Minimum 4.5:1 ratio
- White text on primary purple: Meets AA standard
- Error text (#dc3545) on white: Meets AA standard

Keyboard Navigation:
- All interactive elements must be keyboard accessible
- Tab order must be logical (top to bottom, left to right)
- Focus states must be visible
- Enter key must trigger primary actions

Form Accessibility:
- All form inputs must have associated labels
- Required fields must be marked with asterisk (*)
- Error messages must be associated with inputs
- Submit button must be clearly labeled

Interactive Element Requirements:
- Minimum touch target size: 44x44 pixels
- Adequate spacing between clickable elements
- Clear visual feedback for all interactions

Screen Reader Support:
- Semantic HTML elements used throughout
- Alt text for any images (not applicable in basic version)
- ARIA labels where necessary
- Meaningful page title
